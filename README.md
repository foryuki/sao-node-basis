# sao-cloud

> my awesome SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao cloud my-project
```

### From git

```bash
sao foryuki/sao-cloud my-project
```
